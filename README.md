# Loan Approval Prediction System

## Overview
A Machine Learning classification application that predicts whether a loan application is approved based on applicant information.

## Features
- Loan approval prediction
- Data preprocessing
- Logistic Regression model
- Interactive Streamlit application
- Confidence score prediction

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit

## Machine Learning Workflow
- Data Cleaning
- Label Encoding
- Feature Scaling
- Model Training
- Model Evaluation
- Prediction

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Project Structure
```
loan-approval-prediction/
│── app.py
│── train_model.py
│── requirements.txt
│── model.pkl
│── dataset.csv
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
